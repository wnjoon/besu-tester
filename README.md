# Hyperledger Besu 

## How To Run

- run.sh
    - Run default Besu node with input parameter value of node number
    - ```$ ./run.sh 1```
- run-privacy.sh
    - Run privacy-enabled Besu node with input parameter value of node number
    - Should [run tessera node](./tessera/README.md) before run this script
    - ```$ ./run-privacy.sh 1```
